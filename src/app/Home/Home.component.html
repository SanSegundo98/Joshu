About Joshu:

Joshu (tries) to help you learn japanese. Making use of mnemonics, the objective is to allow you to more easily remember certain words you might struggle to remember.

For example, 何方 would be equal to 'Donatello', like the sculptor (or the turtle, if you prefer).

To start using Joshu, just sign up using the form to the left, or login if you are already signed up, and then add a few cards.

Happy learning. 頑張って！
